# Basic Portfolio

Simple static portfolio prototype built with:

- HTML
- Bulma
- a small custom stylesheet
- a little vanilla JavaScript for the UX sample viewer

## Open locally

You can open `index.html` directly in a browser, or serve the folder:

```bash
cd /Users/jremington/Desktop/Coding/basic-portfolio
python3 -m http.server 8000
```

Then open:

- http://localhost:8000
